# NLPToolKit
自然语言处理中常用的工具箱函数,支持开箱即用。


## [功能函数](utils.py)(完善中...)
- 编辑距离(带条件)
- 判断是否为汉字
- 判断是否为数字
- 判断是否为英文
- 全角转半角
- 半角转全角
- 格式化字符串
- 繁体转简体
- 取除同音字
- 取出连续子串
- 取列表中指定元素的所有下标
- ...
